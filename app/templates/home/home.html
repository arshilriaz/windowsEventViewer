Enter Machine Data

Form Instructions

Here are some guidelines for filling out the form:

  • Hostname: Enter the unique network name of the machine. Easiest way to find it is open Terminal/ Powershell in Administrator mode and enter 'hostname' command for getting local logs and enter ipconfig to get IP Address to get remote logs.
  • Username & Password: Provide your credentials for authentication. Ensure your password is strong to prevent unauthorized access. Open Terminal/ Powershell in Administrator mode and enter 'net user' to find admin username.
  • Confirm Password: Re-enter your password to avoid mistakes.
  • Upload EVTX File: If applicable, check the box and select the .evtx file you wish to upload. This file should contain system event logs for troubleshooting and analysis.

Ensure all information is correct before submitting the form to avoid errors in data processing.
